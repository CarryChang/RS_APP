#### RS_APP

一个基于豆瓣评论数据的电影推荐系统应用实例


#### 场景梳理
   1. 首页 Feed 流个性化推荐
   2. 相关用户推荐
   3. 下一个电影推荐
   4. 物品 Push 用户
   5. 基于标签相似度推荐


算法使用
1. FM Embedding 解决评分预测问题
2. DeepWalk Embedding 解决用户序列数据的推荐
3. Content Based 解决电影标签推荐
